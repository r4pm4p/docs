# descritiva de todas as funcionalidaes

## usuario não cadastrado

1. visualizar o mapa.
2. ver eventos.

## usuario

1. ver participantes do evento
2. indicar que ira comparecer ao avento ( assistir evento )

## musico

1. criar vulgo
2. adicionar links de soundcloud, spotify, youtube
3. criar grupo/guilda/tribo
4. criar evento
5. solicitar participação em evento
6. caso seja o criador do evento, pode aceitar ou não quem participa

## representante de arena

1. criar evento ( precisa estar vinculado com arena vinculada )
2. aceitar musicos que desejam participar
3. criar o chaveamento
4. gerenciar o chavemanto


## chaveamento

1. o chaveamento é gerado a partir de um tipo de chavemanto ( Ex.: 8final, 16final)
2. Vinculado com 1 evento, e tendo a regra definida do numero que pode existir de confrontos

## podio

1. Criado a partir de uma evento, tendo como posicionamento 3 musicos
